# 替换应用图标

## 更新内容

为应用添加了自定义图标支持，包括打包后的应用图标和运行时的窗口图标。

## 修改文件

### 1. package.json

在 `build` 配置中为各平台添加了图标路径：

```json
"mac": {
  "icon": "build/icon.icns"
},
"win": {
  "icon": "build/icon.ico"
},
"linux": {
  "icon": "build/icon.png"
}
```

### 2. src/main/main.ts

在 `createWindow()` 函数中添加了窗口图标设置：

```typescript
const iconPath = process.platform === 'win32'
  ? path.join(__dirname, '../../build/icon.ico')
  : path.join(__dirname, '../../build/icon.png');

mainWindow = new BrowserWindow({
  // ...其他配置
  icon: iconPath,  // 设置窗口图标
  // ...
});
```

这样可以确保：
- Windows 使用 `.ico` 格式图标
- macOS 和 Linux 使用 `.png` 格式图标

### 3. build/ 目录

创建了 `build/` 目录用于存放图标文件：
- `icon.icns` - macOS 应用图标
- `icon.ico` - Windows 应用图标
- `icon.png` - Linux 应用图标

## 使用说明

1. 将准备好的图标文件放入 `build/` 目录
2. 运行打包命令：
   ```bash
   npm run package:mac   # macOS
   npm run package:win   # Windows
   npm run package:linux # Linux
   ```
3. 打包后的应用将使用新图标

## 图标要求

- macOS (.icns): 512x512 像素
- Windows (.ico): 256x256 像素
- Linux (.png): 512x512 像素

建议使用高质量的正方形图片（1024x1024 像素）作为源文件，然后使用在线工具转换为所需格式。
