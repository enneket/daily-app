# v1.0.1 版本发布

## 🎉 发布说明

日报桌面应用 v1.0.1 版本发布！这是 v1.0.0 正式版后的第一个更新版本，主要修复了配置和渲染问题。

## 🐛 修复内容

### 1. 动态配置支持（v1.2.5）

**问题**：VitePress 配置中的仓库信息是写死的，不同用户需要手动修改。

**解决**：
- 引入占位符系统（`{{REPO_OWNER}}`、`{{REPO_NAME}}`）
- 初始化时自动替换为用户的实际配置
- 支持动态生成 GitHub 链接、base 路径、sitemap 等

**影响文件**：
- `site/.vitepress/config.ts` - 使用占位符
- `scripts/generate-index.js` - 使用占位符
- `src/main/github-service.ts` - 添加替换逻辑

### 2. 统计页面渲染修复（v1.2.6）

**问题**：统计页面显示原始 HTML 代码而不是渲染后的内容。

**解决**：
- 重写统计页面结构
- 使用 frontmatter 和 Markdown 标题
- 优化数据处理逻辑
- 改进 Vue 模板语法

**影响文件**：
- `site/stats.md` - 完全重写
- `src/main/github-service.ts` - 更新版本号

## ✨ 改进内容

### 自动化配置

用户只需在桌面应用中配置一次：
- 仓库所有者
- 仓库名称

所有相关配置自动生成：
- VitePress base 路径
- GitHub 图标链接
- Sitemap 主机名
- 日报索引路径

### 更好的用户体验

- ✅ 统计页面正确显示数据
- ✅ GitHub 图标链接到正确的仓库
- ✅ 详情页链接包含正确的 base 前缀
- ✅ 无需手动修改配置文件
