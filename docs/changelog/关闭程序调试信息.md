# 关闭程序调试信息

## 变更内容

移除了程序运行时输出到控制台的调试信息。

## 涉及文件

- `src/main/main.ts`
- `src/main/github-service.ts`

## 具体改动

### src/main/main.ts

- `log()` 函数移除 `console.log(msg)` 调用，日志仅写入文件（`icon-debug.log`）
- 移除启动时的 `=== Debug Info ===` 调试块输出
- 移除 `console-message` 事件监听（renderer 控制台转发）
- 移除定时检查触发自动提交时的 `console.log`
- 移除 Linux 图标设置失败时的 `console.log`

### src/main/github-service.ts

移除以下方法中的所有 `console.log` 和 `console.warn`：

- `checkVersion()` - 版本检查日志
- `cleanupRemovedFiles()` - 文件清理日志
- `migrateOldReports()` - 文件迁移日志
- `updateRepoFiles()` - 文件更新日志
- `shouldAutoPush()` - 自动推送判断日志
- `submitReport()` - 重复内容检测日志

## 保留内容

保留所有 `console.error` 用于错误上报，方便排查问题。
