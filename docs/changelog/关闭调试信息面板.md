# 关闭调试信息面板

## 问题描述
`MainEditor.tsx` 中存在调试信息面板，在用户提交日报时会显示详细的调试日志，不应在生产环境中展示给用户。

## 修改内容

### `src/renderer/components/MainEditor.tsx`
- 移除 `debugInfo` 状态（`useState<string[]>([])`）
- 移除 `initialized` 状态及其 `useEffect`
- 移除 `addDebugInfo` 方法
- 移除 `handleSubmit` 中所有 `addDebugInfo(...)` 调用
- 移除 JSX 中的"调试信息"面板渲染块

## 结果
用户提交日报时不再显示调试信息面板，界面更简洁。
