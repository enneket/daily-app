# v1.0.4 版本发布

**发布日期**：2026-03-21

## 版本亮点

- **测试体系建设**：引入 Vitest 单元测试框架
- **核心功能覆盖**：crypto.ts、constants.ts、github-service.ts 单元测试
- **代码质量提升**：28 个测试用例全部通过

- 添加 `@vitest/coverage-v8` 覆盖率报告
- 新增 `npm test` 和 `npm run test:run` 命令
