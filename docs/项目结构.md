# 项目结构

## 目录结构

```
daily-app/
├── src/
│   ├── main/                    # Electron 主进程 (Node.js)
│   │   ├── main.ts            # 主进程入口
│   │   ├── preload.ts          # 预加载脚本
│   │   ├── github-service.ts   # GitHub API 服务
│   │   ├── crypto.ts           # Token 加密/解密
│   │   ├── constants.ts         # 常量配置
│   │   ├── service-locator.ts  # 服务定位器
│   │   ├── logger.ts           # 日志模块
│   │   ├── types.ts            # 类型定义
│   │   └── *.test.ts           # 单元测试
│   │
│   └── renderer/                # React 前端 (浏览器)
│       ├── App.tsx             # 应用根组件
│       ├── main.tsx            # 渲染进程入口
│       ├── index.css           # 全局样式
│       ├── types.ts            # 前端类型定义
│       └── components/
│           ├── MainEditor.tsx  # 主编辑器组件
│           └── Settings.tsx     # 设置页面组件
│
├── site/                        # 日报展示网站模板 (VitePress)
│   ├── .vitepress/
│   │   ├── config.ts          # 网站配置
│   │   ├── reports-index.data.ts
│   │   ├── stats.data.ts
│   │   └── *.json
│   └── *.md                    # 网站页面
│
├── scripts/                     # 构建脚本
│   └── generate-index.js       # 日报索引生成
│
├── docs/                        # 项目文档
│
├── dist/                        # 构建输出 (自动生成)
├── release/                     # 打包输出 (自动生成)
└── build/                       # 应用图标等资源
```

## 核心模块

### 主进程 (src/main/)

| 文件 | 说明 |
|------|------|
| main.ts | Electron 主进程入口，创建窗口、处理 IPC |
| preload.ts | 预加载脚本，暴露安全的 API 给渲染进程 |
| github-service.ts | GitHub API 封装，处理日报提交、仓库初始化 |
| crypto.ts | Token 加密存储（使用 Electron safeStorage） |
| constants.ts | 提交策略、API 配置常量 |
| service-locator.ts | GitHub Service 单例管理 |
| logger.ts | 日志模块 |
| types.ts | 主进程类型定义 |

### 渲染进程 (src/renderer/)

| 文件 | 说明 |
|------|------|
| App.tsx | 应用根组件，路由和状态管理 |
| MainEditor.tsx | 主编辑器界面 |
| Settings.tsx | 设置页面 |
| types.ts | 前端类型定义 |

### 网站 (site/)

VitePress 静态网站生成器，用于展示日报。

## 配置文件

| 文件 | 说明 |
|------|------|
| package.json | 项目配置、依赖、脚本命令 |
| tsconfig.json | 渲染进程 TypeScript 配置 |
| tsconfig.main.json | 主进程 TypeScript 配置 |
| vite.config.ts | Vite 构建配置 |
| vitest.config.ts | Vitest 测试配置 |
| tailwind.config.js | Tailwind CSS 配置 |
| electron-builder.yml | Electron 打包配置 |

## 测试文件

| 测试文件 | 覆盖模块 |
|----------|----------|
| constants.test.ts | 常量 getter 函数 |
| crypto.test.ts | Token 加密/解密/存储/迁移 |
| github-service.test.ts | GitHub 服务核心逻辑 |

运行测试：

```bash
npm test        # 监听模式
npm run test:run  # 单次运行
```
