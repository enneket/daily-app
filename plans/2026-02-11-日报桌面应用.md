# 日报桌面应用开发计划

**日期**: 2026-02-11  
**需求**: 跨平台日报桌面应用

## 需求概述

开发一个 Mac 和 Windows 通用的桌面应用，用于编写和提交日报到 GitHub 仓库。

## 核心功能

### 1. 主界面
- 大的文本输入框（支持多行输入）
- 右下角：提交按钮
- 左侧：设置按钮

### 2. 设置功能
- GitHub 个人访问令牌（PAT）配置
- 目标仓库配置（用户名/仓库名）
- 分支配置（默认 main）

### 3. 提交功能
- 支持同一天多次提交
- 自动追加到当天的日报文件
- 提交后清空输入框

### 4. GitHub 存储结构
```
YYYY/
  MM/
    DD.md
```

示例：
```
2026/
  02/
    11.md
    12.md
  03/
    01.md
```

## 技术选型

- **框架**: Electron（跨平台桌面应用）
- **前端**: React + TypeScript
- **样式**: Tailwind CSS
- **GitHub API**: Octokit
- **构建**: electron-builder
